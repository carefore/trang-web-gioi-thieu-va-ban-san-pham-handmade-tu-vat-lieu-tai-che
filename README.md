
# Handmade Products Website

This repository contains the code for a website showcasing and selling handmade products made from recycled materials.

## Description

This project is a simple web application developed using HTML, PHP, CSS, JavaScript, and jQuery. It allows users to view and add handmade products to the website. Product information such as name, description, and price can be submitted via a form and stored in a database.

## Website Link

Visit our website: [Handmade Products](https://suanon.com.vn/san-pham/colos-iggold)

## How to Use

To use this project, simply clone the repository to your local machine and set up a local web server environment to run the PHP files. Make sure to configure the database connection details in the PHP files before running the application.

### Prerequisites

- Web server (e.g., Apache, Nginx)
- PHP
- MySQL or another SQL database

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/handmade-products.git
```

2. Set up a local web server environment.

3. Configure the database connection details in the PHP files (`save_product.php`).

4. Open the website in a web browser.
